## Installation
To get a local copy working, follow these steps.

Clone the repository
```sh
git clone https://github.com/your_username_/Project-Name.git
```

Install packages
```sh
npm install
```

Modify .env 'DB' value to your own database
```
DB={Add your databsae here}
```

Start server
```sh
npm start
```
